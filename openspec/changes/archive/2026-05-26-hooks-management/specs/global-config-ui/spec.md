## REMOVED Requirements

### Requirement: Global Hook Fields

**Reason**: Global hooks (`runOnConnect`, `runOnConnectRestart`, `runOnDisconnect`) are moved to the new dedicated "Hooks" tab for centralized hooks management.

**Migration**: Users will configure global hooks in the "Hooks" tab instead of the "Cấu hình" (Global Config) tab. The Global Config tab will retain only General Settings and Hot Reload UX.
